## Local Develop

下载最新的稳定版 Node.js

命令行输入 npm 查看是否安装成功

命令行输入 npm install -g yarn

cd 到本项目 （ 本目录下包含 package.json ）

命令行输入 yarn install 下载并安装依赖

yarn start 启动开发预览服务，会自动打开浏览器浏览 localhost:3000

## Publish Updates

命令行输入 yarn build （较为费内存）

使用 SCP 指令进行代码上传

scp -r ./build deai:/home/ubuntu/homepage/
